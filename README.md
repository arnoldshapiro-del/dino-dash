# Dino Dash v3

Premium endless-runner + themed-levels mashup of Chrome Dino + Geometry Dash + 10 classic games.

**Status:** WIP — building 12 phases sequentially.

## Tech
- HTML5 Canvas 2D + vanilla ES6 modules
- Web Audio API for procedural music + SFX
- localStorage for persistence
- Service Worker for offline PWA play
- Zero runtime dependencies

## Build
None — pure static site. Open `index.html` or deploy folder to any static host.

Built with vibe coding via Claude Code.
